    # Klayr colors

    This is a representation of the colors used in the Klayr design system.<br/>
    The colors are divided into 3 categories: light colors, dark colors and grayscale colors.<br/>
    Each color has a name and a hex code.<br/>
    Click on the color square to copy the hex code to your clipboard.<br/>

    ### Tailwind classes
    The colors are also available as tailwind class suffixes. The class names are the color names in camel case and with any numbers connected by a dash `-`.<br/>
    For example, the color `Volt Dark` has the class name `voltDark` and the color `Gray 1` has the class name `gray-1`.<br/>
    A full class name would then be `bg-voltDark` or `text-gray-1`.<br/>

    ### CSS variables
    The colors are also available as CSS variables. The variable names are prefixed with `--color-` and has the color names in camel case and with any numbers connected by a dash `-`.<br/>
    For example, the color `Volt Dark` has the variable name `--color-voltDark` and the color `Gray 1` has the variable name `--color-gray-1`.<br/>
